# SimpleWeather

Egy modern, letisztult **időjárás alkalmazás**, React segítségével, OpenWeatherMap API-val.

![Screenshot](./public/screenshot.png)

---

## Funkciók

- Város alapú időjárás keresés
- Keresés Enter lenyomásra
- Nappal / éjszaka háttér (API alapján)
- Hőmérséklet színezése
- Manuális sötét mód (mentve)
- Utolsó város mentése
- Keresési előzmények (localStorage)
- Előzmények törlése
- Animált ikonok és átmenetek
- Mobilbarát megjelenés

---

## Technológiák

- React
- OpenWeatherMap API
- CSS (responsive, animációk)
- LocalStorage

---

## Telepítés

```bash
git clone https://github.com/doomhyena/simpleweather.git
cd weather-app
npm install
```

### Környezeti változók

Hozz létre egy `.env` fájlt:

```env
VITE_WEATHER_API_KEY=AZ_API_KULCSOD
```

### Indítás

```bash
npm start dev
```
